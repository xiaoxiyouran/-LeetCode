031 下一个排列

链接和考点

LeetCode https://leetcode-cn.com/problems/next-permutation/description/
考点 vector, number
难度 Medi


题意

实现获取下一个排列的函数，算法需要将给定数字序列重新排列成字典序中下一个更大的排列。

如果不存在下一个更大的排列，则将数字重新排列成最小的排列（即升序排列）。

必须原地修改，只允许使用额外常数空间。

以下是一些例子，输入位于左侧列，其相应输出位于右侧列。 1,2,3 → 1,3,2 3,2,1 → 1,2,3 1,1,5 → 1,5,1


分析

1- 对于序列 1 2 3 4 7 3 5 4 2 1 首先找到第一个从后往前变小的元素，这个是倒数 第一个 3；

2- 由于3是从后往前第一个降序的元素，也就是说 5 4 2 1 （3之后）应该都是升序排列。那么比3 略微大一点的元素就是第一个比它小（2）后面的元素(4).

也就是下一个数，应该把倒数第一个3替换成4，然后 5 3 2 1 重新升序排列即可。 （降序改升序，只需要两两互换）；

3- 当都是前面比后面的大时，说明已经是最大排列了，整个翻转即可。

#include "GlobalHead.h"
#include <map>
using namespace std;

class Solution{
public:
	void nextPermutation(vector<int>& num) {
		if(num.empty() || num.size()==0)
			return;
		int i = num.size()-2;
		while(i>=0 && num[i]>=num[i+1])   // 从倒数第2个元素开始找，逆序不是递增的元素
		{
			i--;
		}
		if(i>=0)
		{
			int j=i+1;
			while(j<num.size() && num[j]>num[i])    // 不满足的时候，找到第一个比前面那个元素小的
			{
				j++;
			}
			j--;    //  
			general::print_Vec1D(num);
			int temp = num[i];  // 交换这两个元素
			num[i] = num[j];
			num[j] = temp;
			general::print_Vec1D(num);
		}
		reverse(num, i+1);      // 交换后，原先不是逆序递增元素的后面，都倒叙排列
		general::print_Vec1D(num);
	}
	void reverse(vector<int>& num, int index)
	{
		int l = index;
		int r = (int)num.size()-1;
		while(l<r)              // 左右两边对着交换次序
		{
			int temp = num[l];
			num[l] = num[r];
			num[r] = temp;
			l++;
			r--;
		}
	}
};

int main() {

	vector<int> in{1,2,3,4,7,3,5,4,2,1};
	Solution().nextPermutation(in);
	general::print_Vec1D(in);

	return 0;
}/* output

1 2 3 4 7 4 5 3 2 1 
1 2 3 4 7 4 1 2 3 5 
1 2 3 4 7 4 1 2 3 5 

*/
