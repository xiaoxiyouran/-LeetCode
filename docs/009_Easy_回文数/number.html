009 回文数

链接和考点

LeetCode https://leetcode-cn.com/problems/palindrome-number/description/
考点 number
难度 Easy


题意

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true

示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。

进阶:

你能不将整数转为字符串来解决这个问题吗？


分析

1- 首先排除负数， 和末尾有0的数，它们肯定都不是回文数；

2- 如果是“1221”， 那么从右到左，

rev 1 12
x 122 12

如果是 “121”， 那么从右到左，

rev 1 12
x 12 1

这时候只要判断到两边的是回文数即可。


参考答案

C++ 程序如下：

class Solution {
public:
	bool isPalindrome(int x) {
		 if (x<0 || (x!=0 && x%10==0)) return false;
		int rev = 0;
		while (x>rev){  // 判断到中间即可
			rev = rev*10 + x%10;
			x = x/10;
		}
		return (x==rev || x==rev/10);
	}
};
