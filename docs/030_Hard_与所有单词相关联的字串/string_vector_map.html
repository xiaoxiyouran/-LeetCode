030 与所有单词相关联的字串

链接和考点

LeetCode https://leetcode-cn.com/problems/substring-with-concatenation-of-all-words/description/
考点 string,vector, map
难度 Hard


题意

给定一个字符串 s 和一些长度相同的单词 words。在 s 中找出可以恰好串联 words 中所有单词的子串的起始位置。

注意子串要与 words 中的单词完全匹配，中间不能有其他字符，但不需要考虑 words 中单词串联的顺序。

示例 1:

输入:
  s = "barfoothefoobarman",
  words = ["foo","bar"]
输出: [0,9]
解释: 从索引 0 和 9 开始的子串分别是 "barfoor" 和 "foobar" 。
输出的顺序不重要, [9,0] 也是有效答案。

示例 2:

输入:
  s = "wordgoodstudentgoodword",
  words = ["word","student"]
输出: []


分析

由于题目是长度相同的单词，以单词的长度开始，依次查找。

建立左右两个指针，构成一个窗口， 这个窗口可能包含多个重复的元素，这时将左指针向右移动，直到窗口里的单词的个数依次等于目标查找里的个数；

第一步是检查是否是候选单词， 第二步是检查窗口里的单词没有多的，第三步是在第二步保证没有多余的重复单词的前提下，这时窗口的长度恰好等于需要组合的单词长度，即为找到了一个组合。

class Solution{
public:
	vector<int> findSubstring(string S, vector<string> &L) {
		vector<int> res;
		map<string, int> traits;

		if (L.empty() || S.empty()) return res;

		for (auto s : L)
			traits[s]++;
		int len = L[0].length();

		for (int i = 0; i < len; i++) {
			map<string, int> window;

			int l = i;
			int r = i;
			while (r < S.length()) {
				// step 1
				while (r < S.length()) {
					string word = S.substr(r, len);
					r += len;
					if (traits.find(word) == traits.end()) {    // 如果没有这个字符串
						window.clear();                         // 清空窗口重新来
						l = r;
					}
					else {
						window[word]++;                         // 又开始存有相同的字符串
						if (window[word] >= traits[word])       // 如果个数足够【但也有可能出现次数太多了】，比如两个连续foofoo,跳到第3步，可能构成一组解
							break;
					}
				}

				// step2: 缩小范围
				while (l < r) {
					string head = S.substr(l, len);
					string tail = S.substr(r - len, len);
					if (window[tail] == traits[tail])           // 要重复一定是尾部重复（因为新来的就是在尾部啊【可以假设刚开始只有我一个，肯定是没有重复的】）foo bar foo(重复) or foo bar bar(bar重复，l会移动两次)
						break;
					l += len;
					window[head]--;
				}

				// step3: 检查是否是一组解
				if (r - l == len * L.size()) {
					res.push_back(l);
				}
			}
		}

		return res;
	}

};
