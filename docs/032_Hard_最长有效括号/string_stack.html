032 最长有效括号

链接和考点

LeetCode https://leetcode.com/submissions/detail/119821580/
考点 string, stack
难度 Hard


题意

给定一个只包含 '(' 和 ')' 的字符串，找出最长的包含有效括号的子串的长度。

示例 1:

输入: "(()"
输出: 2
解释: 最长有效括号子串为 "()"

示例 2:

输入: ")()())"
输出: 4
解释: 最长有效括号子串为 "()()"


分析

1- 对于序列 "(()”， 用一个栈来保存左括号， 只要是左括号就压栈， 碰到一个右括号就弹栈。如果弹栈之后，

  • 栈刚好为空， 那说明刚好是 “()” 配对的情形；
  • 如上， 这个栈不为空， 那么就是栈顶元素后面一个位置开始的（因为配对的已经事先弹出了）

2- 对于序列 ")()())” , 如果是，

  • 右括号，且栈为空，说明不构成配对， 下次开始记录的左括号索引从下一个字符开始。
  • 右括号，且栈为非空，说明构成配对， 弹栈之后（回到步骤1）：

​ 1） 栈空， 刚好是 “()” 配对的情形；

​ 2） 栈非空， 栈顶元素后面一个位置开始的（因为配对的已经事先弹出了）


参考答案

答案如下：

C++ 版本如下：

#include <iostream>
#include <vector>
#include <map>
#include <stack>

using namespace std;

class Solution
{
public:
	int longestValidParentheses(string s)
	{
		int res = 0, l = 0;
		stack<int> si;
		for(int i = 0; i < s.size(); ++ i)
		{
			if(s[i] == '(')
				si.push(i);         // 把左括号的序号压入
			else                    // s[i] == ')'
			{
				if(si.empty())      // if s[i] == ')', 并且栈已空， 那么当前肯定是打断了连续的括号，需要重新记录开始起点
					l = i + 1;      // 栈清空了之后，下一轮要重新开始。对应此时的序号，只数连续的括号
				else
				{
					si.pop();
					if(si.empty())
						res = max(res, i - l + 1);      // 要求是连续的()()()对数最多
					else
						// process "(()" 
						res = max(res, i - si.top());   // 注意栈顶元素已经弹出了一个，这里是剩下的，所以i-si.top() 是不加1的
				}
			}
		}
		return res;
	}
};

int main() {
	string in;
	cin >> in;
	cout << Solution().longestValidParentheses(in) << endl;

	return 0;
}
